here is a list of the supported project's platforms.
Some platforms needs to run an exe file in order to run correctly.

=== Web ===
No need to run anything.

=== Mobile === 
An andriod phone must be connected to the usb port.

=== API ===
Run <project folder>\grafana-7.5.7\bin\grafana-server.exe

=== Electron ===
No need to run anything.

=== WinAPP ===
Run <PROGRAM FILES 86X>\Windows Application Driver\WinAppDriver.exe
